# HAP Convert README

서버에 설치되는 유틸리티 프로그램으로 HAP 코덱 변환 및 미리보기를 지원하는 툴킷이다.
코덱 변환은 ffmpeg 라이브러리를 활용하였다.

## Features

- 미디어 파일 정보 표출
- 미디어 파일 열기 및 저장
- HAP / HAP-Q / HAP-Alpha 코덱 지원
- 변환파일 미리보기
- 변환옵션 저장 및 불러오기
