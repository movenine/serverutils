파일 작업

파일 열기 / 저장


1. 파일 열기 버튼을 누르면, Select a video file 창이 나타나고 *.mp4 *.avi *.mkv 등의 비디오 파일 확장자 파일만 선택할 수 있습니다.



2. 파일 저장 버튼을 누르면, Save a video file 창이 나타나고, 저장할 파일 이름을 입력합니다. (파일형식은 mov)



이미 있는 파일을 선택할 경우, 아래와 같이 안내창이 나타나고 다시 저장할 수 있습니다.



3. 파일 열기를 성공하면, 파일정보 창에 소스파일의 기본 정보가 표시됩니다.



옵션 파일 가져오기 / 내보내기


1. 메뉴바-파일-옵션 가져오기 버튼을 누르면, Import a option file 창이 나타나고 저장된 변환 시나리오 옵션파일(*.opt)을 가져옵니다.



옵션파일을 가져오기 하면, 메인 창의 포멧 변환 옵션 값들이 모두 반영됩니다.



*예시 {A스튜디오_시나리오2_3840x2160_30fps_HAP_ALPHA.opt} 옵션파일 내용*

  → opt 파일을 메모장으로 열고 값을 편집하여 저장할 수 있으며, 저장된 파일을 가져올 수 있습니다. (단, 수치정도만 수정할 것을 권장)




2. 메뉴바-파일-옵션 내보내기 버튼을 누르면, Export a option file 창이 나타나고 입력한 파일이름으로 옵션파알(*.opt)을 내보내기 합니다.
