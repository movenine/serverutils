    설명서

    • 개요


    4K(UHD) 해상도 이상의 고해상도 파일의 원활한 재생 성능을 위하여 HAP 코덱을 권장하고 있어, 대부분의 미디어 플레이어 프로그램들이 고해상도 재생을 위해  HAP 코덱을 지원합니다.

    이러한 HAP 코덱을 사용하기 위해서는 기존의 압축된 영상파일(H.264/265)을 변환해 주어야 합니다. 단, HAP 코덱파일은 무압축 방식이므로 파일 크기가 5배이상 커지므로 파일 관리에 유의 하여야 합니다.

    또한 일부 미디어 플레이어는 FFmpeg 하드웨어 가속기능을 지원하는데, HAP 코덱파일은 호환됩니다.


    사용자 인터페이스


    HAP Convert은 윈도우 작업표시줄 Tray 아이콘을 더블클릭하거나 우클릭-HAP코덱변환으로 실행된다.

    사용방법, 프로그램정보 

    1

    파일열기 버튼

    변환할 소스파일의 경로를 선택

    2

    파일저장 버튼

    저장할 경로 및 파일명을 입력하여 저장

    3

    미리보기 버튼

    [파일열기]/[파일저장] 경로에 있는 파일을 재생하고, 변환 후의 상태를 미리 확인할 수 있음

    4

    파일정보 뷰

    지정된 파일의 코덱, 비트레이트, 비율, 프레임 등의 정보를 표시

    5

    가로해상도 입력

    변환할 가로 해상도를 입력할 수 있는 텍스트 창

    6

    세로해상도 입력

    변환할 세로 해상도를 입력할 수 있는 텍스트 창

    7

    코덱 선택

    변환할 코덱을 미리 지정한 리스트에서 선택

    8

    인코딩 코어수 입력

    인코딩에 사용할 CPU의 코어수를 지정할 수 있음 (단, 최대 코어수를 넘지 못함)

    9

    Scale 옵션 활성화 체크

    체크하면 Scale 옵션을 사용가능

    10

    원본 해상도 체크

    소스파일의 원본 해상도를 그대로 유지하고 변환

    11

    종횡비 고정 체크

    체크하면 현재 비율을 유지하고, 가로해상도 대비 세로해상도를 자동으로 맞춤

    12

    Scale 알고리즘 선택

    Default : lanczos 

    해상도 Scale 보정을 할 때 적용되는 색보정에 대한 알고리즘을 제공

    13

    Add Script 입력

    FFmpeg 라이브러리에서 제공하는 옵션을 사용자가 직접 명령어를 입력하여 적용이 가능 (FFmpeg Documents 링크 참조)

    14

    변환 버튼

    파일 변환 시작

    15

    변환 진행바

    변환되는 진행상태를 표시 (~100%)



    • 파일 열기 / 저장


    1. 파일 열기 버튼을 누르면, Select a video file 창이 나타나고 *.mp4 *.avi *.mkv 등의 비디오 파일 확장자 파일만 선택할 수 있습니다.



    2. 파일 저장 버튼을 누르면, Save a video file 창이 나타나고, 저장할 파일 이름을 입력합니다. (파일형식은 mov)



    이미 있는 파일을 선택할 경우, 아래와 같이 안내창이 나타나고 다시 저장할 수 있습니다.



    3. 파일 열기를 성공하면, 파일정보 창에 소스파일의 기본 정보가 표시됩니다.



    옵션 파일 가져오기 / 내보내기


    1. 메뉴바-파일-옵션 가져오기 버튼을 누르면, Import a option file 창이 나타나고 저장된 변환 시나리오 옵션파일(*.opt)을 가져옵니다.



    옵션파일을 가져오기 하면, 메인 창의 포멧 변환 옵션 값들이 모두 반영됩니다.



    *예시 {A스튜디오_시나리오2_3840x2160_30fps_HAP_ALPHA.opt} 옵션파일 내용*

      → opt 파일을 메모장으로 열고 값을 편집하여 저장할 수 있으며, 저장된 파일을 가져올 수 있습니다. (단, 수치정도만 수정할 것을 권장)




    2. 메뉴바-파일-옵션 내보내기 버튼을 누르면, Export a option file 창이 나타나고 입력한 파일이름으로 옵션파알(*.opt)을 내보내기 합니다.






    • 코덱 변환


    앞서 변환할 파일 열기/저장 및 옵션파일 작업이 완료되면, 메인 창 하단의 변환 버튼을 클릭합니다.



    변환 진행바에 백분율(%) 표시되며, 완료되면 파일정보창에 출력파일정보와 함께 정상적으로 완료되었다는 "변환완료!" 상태메세지가 표시됩니다.

    변환오류




    • 미리보기


    코덱 변환이 완료되고 미리보기 버튼을 누르면, 메인 창 기준 좌측에 소스파일과 출력파일이 동시에 재생됩니다.

    미리보기 창은 고정비율로 크기조절이 가능하고 키보드 좌(←), 우( →) 키를 통해 앞으로/뒤로 감기 제어가 가능합니다.





    • 사용자 스크립트


    Add Script 텍스트 박스에 커맨드 입력을 통해서  FFmpeg에서 제공하는 수 많은 변환 옵션을 적용할 수 있습니다.



    우측에 있는 FFmpeg Documents를 클릭하면, 공식 사이트로 접속하여 더 많은 정보를 확인할 수 있습니다.



    유용한 스크립트 예제


    프레임 레이트 강제설정 (30, 60, 29.97, 59.94 ...)

    -r 30

     

    소스파일 전체해상도의 중심을 기준으로 3분의 2크기로 좌/우를 Crop (ex. 15360 x 2160 -> 10240x2160)

    -vf crop=2/3*in_w:3/3*in_h


    소스파일 좌측부터 전체 해상도의 절반크기로 Crop

    -vf crop=in_w/2:in_h:0*in_w:in_h


    Contrast 를 증가

    -vf "colorlevels=rimin=0.039:gimin=0.039:bimin=0.039:rimax=0.96:gimax=0.96:bimax=0.96"
